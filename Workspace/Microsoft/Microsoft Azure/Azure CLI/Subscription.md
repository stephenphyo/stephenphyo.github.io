# Microsoft Azure CLI

## Subscription

### **<u>Account Configuration</u>**

#### Login to Account

```
az login
```

#### List All Accounts

```
az account list
```

#### Set Specific Subscription

```
az account set --subscription="<subscription_id>"
```

### View Current Subcription

```
az account show
```

#### Get Azure Regions

```
az account list-locations -o table
```
